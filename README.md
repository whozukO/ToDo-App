# Todo App

Простое веб-приложение для управления задачами.

## 🚀 Demo

[Открыть проект](https://whozuko.github.io/ToDo-App/)

## 🛠 Технологии

- Vanilla JavaScript
- SCSS
- Vite

## 📦 Установка и запуск

```bash
git clone https://github.com/whozukO/ToDo-App.git
cd ToDo-App
npm install
npm run dev
```

## ✨ Функциональность

- Добавление задач
- Удаление задач
- Редактирование задачи
- Отметка задачи как выполненной
- Фильтрация задач по статусу — все, активные, выполненные
- Счётчик активных задач
- Сохранение задач в localStorage
- Переключение темы
